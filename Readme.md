# Netflix Landing Page (NetFlip)

## Project Overview

This project is a Netflix-inspired landing page built using HTML and CSS. It demonstrates understanding of semantic HTML, CSS styling, layout structure, and responsive design.

## Features

- Hero section with background image and navigation
- Email signup form
- Video trailer section
- Trending section
- Feature cards section
- FAQ section using HTML `<details>`
- Responsive design for mobile and desktop

## Technologies Used

- HTML5
- CSS3

## Key Learning Outcomes

- Semantic HTML structure
- CSS Flexbox and layout techniques
- Responsive design using media queries
- UI replication skills
- Working with multimedia elements

## How to Run

1. Clone or download the repository
2. Open index.html in a browser
